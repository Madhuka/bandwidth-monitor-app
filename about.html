My Speed!

My Speed is a Web Application that checks the Network Bandwidth.

This web application will calculate your network speed. Please select a file to be downloaded* from any of the given locations. The file will be downloaded automatically to your computer and the speed of the network connection will be shown in the Network speedometer. Information about your test will be kept for future references.

Bandwidth Recommended Test File Size
1 Mbps 1 MB
2Mbps- 10Mbps 10 MB
10Mbps- 20Mbps 35 MB
20Mbps & above 100 MB/500 MB
*Please note that these tests will consume your monthly data capacity
